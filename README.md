Music Guessing Game (Python Desktop App)<br>
A classic music guessing game project developed in Python, focusing on creating a graphical user interface (GUI) and handling pure logic data structures without relying on AI.<br>

Tech Stack<br>
Language: Python 3.x<br>

Library: Tkinter (for GUI creation), Random (for random number generation)<br>

Logic: Binary Tree Implementation / In-order Traversal (for managing the internal music database)<br>

File Structure<br>
superfinalproject.py: The main code file, over 900 lines long, containing all the logic in one place (Monolithic Architecture)<br>

bg.gif / choosetype.gif / bg2.gif / bgforexplain.gif / correctans.gif / find.gif / wrongans.gif : Game graphic files<br>
